- SFI, [Foundations of Intelligence in Artificial and Natural Systems](https://wiki.santafe.edu/index.php/Foundations_of_Intelligence_in_Natural_and_Artificial_Systems) project[^report]
- Solving the problem of meaning by making algorithms able to handle analogies. How does it compare to solving the problem from a behavioural perspective?
- > *So people would create one big data set to train and test their neural network on and publish a paper saying, “Our method gets 80% right on this test.” And somebody else would say, “Wait, your data set has some weird statistical properties that allow the machine to learn how to solve them without being able to generalize. Here’s a new data set that your machine does horribly on, but ours does great.” And this goes on and on and on.*
- Abstraction and Reasoning Corpus
- > *Embodiment could be an essential feature of cognitive agents that are able to do common sense reasoning (the kind of reasoning babies can do)*
