# Jupyter Books
[Official website](http://www.jupyterbook.org)

Jupyter books are a tool from the Jupyter ecosystem that can be used to create HTML, PDF or Markdown books that look nice by default and are publication ready (both physically and online).

Since Jupyter Books are built with code, they can be created collaboratively in a natural way, using standard open source practices. For example, The Turing Way[^turingway] is built as a Jupyter Book.
