# Stability properties of labeling recursive auto-associative memory

<p style="text-align: right">
<a href="
http://ieeexplore.ieee.org/document/471363/
">
<i>
A. Sperduti, 1995</i>
</a>
</p>

***

- underlying concepts: RAAM model by Pollack, Hopfield networks[^hopfield]
- main topic: Labeling Recursive Auto-Associative Memory
- main claims:
	- a LRAAM can be converted in an analog Hopfield network
	- in the analog architecture, data can be accessed by content using a key-value-like storage; giving a "key", can be seen as constraining the network
	- a constrained version of the network has a specific fixed point, for which some sufficient conditions for stability are proved
	- under some conditions, stability conditions for a specific constrained version of the network  can be extended to other *related* constrained versions of the network

***

- the area of the paper is that of encoding networks, i.e. networks which can represent data structures (e.g. lists, trees, graphs) internally, i.e. building representations.
- RAAM networks can represent lists and trees; LRAAM can represent even labeled directed graphs
