One important technical skill needed both for modeling work and for more generic engineering tasks involving deep neural networks is loss function optimization.

Several papers are probably published each year presenting empirical as well as theoretical evidence that some new technique improves learning algorithms in some sense. Keeping updated about these inventions can save time when training models and adds some working tools to one's belt.

One way to keep updated about such results, to select the most important, widely recognized and useful ones, is to check out which ones have been implemented in the most common DL libraries, such as PyTorch or TensorFlow. This is more generally a reflection of where the attention of leading tech research scientists is focused.
