# Linking notes
I noticed that my notes graph structure is a little cluttered and links interpretation is difficult. So I think that to get the most out of the graph structure I should start using some rules of thumb for linking. I can say for sure that until now I create three kinds of links in my slipbox:
1. structure links: they are there whenever I create outline notes that reference to their subparts
2. reference links: they are links to, well... reference notes
3. thinking links: they are the proper connections between notes I write that are not intended to be a part of the same outline. 

The latter should be the ones that are valued the most, but are also the ones in which clutter shows up the most. Removing reference links from the graph view the situation is a bit clearer.
