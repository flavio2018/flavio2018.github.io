# Machine Learning levels of understanding
<p style= "text-align: right"><i>Created 05/11/2021</i></p>

There are probably several levels of understanding of machine learning models; maybe these are comparable to Marr's three levels of analysis of cognitive processes[^marr].

Two of these are surely the computational level and the functional level. The first one is the level at which the mathematical properties and structure of the model should be understood. The second one is the understanding of the high-level properties of the resulting model, such as its memory properties. These can be investigated in several ways.

One additional level is added in practice, which could be called the implementation level. This is the level at which we need to understand the code that implements the mathematical structures in order to get it to work - and possibly to extend it or embed it in other applications.

[^marr]: [[@marr_understanding_1976|From Understanding Computation to Understanding Neural Circuitry]]
